# 仮説

## 変更内容

透過画像からFAX蛍光画像を生成するベースラインモデルを構築する。

- **アーキテクチャ**: U-Net（ResNet34 encoder, ImageNet pretrained）
- **損失関数**: L1 Loss + SSIM Loss
- **データ拡張**: なし（初期）

## 期待する効果

- SSIM: 0.75 以上（最低限の構造再現）
- PSNR: 25 dB 以上（最低限のノイズ抑制）

## 根拠

1. **U-Net**: 医療画像タスクで実績のあるアーキテクチャ
   - Skip connection により入力画像の構造情報を保持
   - Encoder-Decoder構造で効率的な特徴抽出

2. **ResNet34 + ImageNet pretrained**:
   - 低レベル特徴（エッジ・テクスチャ）の転移学習
   - 学習の高速化と安定性向上

3. **L1 + SSIM Loss**:
   - L1: ピクセル単位の誤差最小化
   - SSIM: 構造的類似度の直接最適化（評価指標と一致）

## リスク

- グレースケール→カラー変換ではないため、転移学習の効果が限定的な可能性
- SSIM Lossの重み設定によっては過度にスムージングされる可能性
- 512×512の解像度でメモリ不足が発生する可能性

## ベースラインの目的

1. パイプライン全体の動作確認（データ読み込み→訓練→評価→提出CSV生成）
2. SSIM/PSNRの基準値を確立
3. 今後の改善幅を測定する基準点
4. GPU使用量・訓練時間の見積もり
