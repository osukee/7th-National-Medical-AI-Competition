# 仮説

## 変更内容

学習時の画像解像度を512×512から768×768に増加させる。

- **image_size**: 512 → 768
- **batch_size**: 4 → 2（メモリ制約対応）

## 期待する効果

- **SSIM**: ベースラインから +0.01〜0.02 改善
- **PSNR**: ベースラインから +0.5〜1.0 dB 改善
- 細部の再現性向上（テクスチャ・エッジ）

## 根拠

1. **高解像度入力**: 細かいテクスチャ情報を保持
2. **元画像との距離縮小**: 1920×1440→768は512より元画像に近い
3. **医療画像の詳細**: オルガノイドの微細構造を捉えやすい

## リスク

- メモリ使用量の大幅増加
- 学習時間の増加（約2倍）
- 過学習リスクの増大
